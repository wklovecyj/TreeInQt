# TreeInQt说明文档

## 构成

本项目一共有三个关键点
1. 基本运算函数，负责拆解表达式，塞入栈，构建树
2. 各个点位置计算和返回点列表和边列表
3. qml组件

其中最有意思的是点的位置计算，首先给中序遍历树，然后按中序遍历的顺序，从左到右赋予id，id大小便是各个点在x轴上的相对位置大小，y则用深度表示。然后在构图前获取到显示区域Item到width和height，于是按比例分割，便可做到图像居中且相对位置一致

还有要注意到的点是Qxx和xx的转换，例如QString和String，还有QChar和Char，此处由ai负责
还需要思考的部分是绘图和绘线用的列表是QVariantList类型，，里面装的是QVariantMap类型。
